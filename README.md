# leetcode_cpp

Solutions for leetcode using c++11.

## Compile & Run

-   Compile only:

    `make <prog>`

-   Run test:

    `./build/<prog> -s -d yes`

-   Compile & Test:

    `TEST=1 make <prog>`

-   Add compiler flags:

    `CXXFLAGS=-O2 make <prog>`

-   Run all leetcodes:

    `TEST=1 make all`
